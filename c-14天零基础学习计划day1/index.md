# C++14天零基础学习计划Day1

## 第 1 天：C++ 与 C 的核心差异（基础语法）

### 学习目标

掌握 C++ 的基本语法框架，区分与 C 的核心不同点。

### 核心内容

1. C++ 程序结构：main函数写法、头文件规范（#include <iostream> 而非 <stdio.h>，namespace 命名空间）。
2. 输入输出：std::cout/std::cin（对比 C 的printf/scanf）。
3. 数据类型增强：bool 类型、引用（&，重点！对比 C 的指针）、const 增强（C 的 const vs C++ 的 const）。
4. 函数增强：函数重载（C 不支持，Qt 信号槽底层用到）、默认参数。

### 练习

* 用 C++ 改写 C 的 “Hello World”，添加命名空间、引用传参的函数。
* 写一个支持重载的加法函数（int/float/ string）。
